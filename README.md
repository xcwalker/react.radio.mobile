# react.radio
A lightweight react based website for streaming radio.

### Uses an api link for:
 - [X] Current Song
 - [X] Past Songs
 - [X] DJs
 
### Features:
 - [X] Custom Player
 - [X] Timetable
 
### Requires:
 - API url (Example: https://apiv2.simulatorradio.com)
 - Audio Stream (Example: https://simulatorradio.stream)
